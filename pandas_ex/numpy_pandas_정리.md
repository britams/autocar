# numpy & pandas 코린이 정리

## 1. numpy가 뭐야?

**숫자 배열(리스트)을 빠르게 계산하기 위한 라이브러리**예요.

파이썬 기본 리스트로도 계산할 수 있지만, 데이터가 많아지면 엄청 느려져요. numpy는 이걸 훨씬 빠르게 처리해줘요.

```python
import numpy as np

arr = np.array([1, 2, 3, 4])
print(arr * 2)   # [2 4 6 8]  -> 리스트라면 반복문 돌려야 하는데 numpy는 바로 됨
```

### 언제 쓰나?
- 이미지 데이터 다룰 때 (이미지 = 숫자 배열이에요)
- 행렬 계산, 벡터 계산 할 때
- 대량의 숫자 데이터를 빠르게 처리할 때

### uint8 같은 자료형도 numpy가 관리해요
- `uint8`: 0~255 (이미지 픽셀값에 딱 맞음, [uint8 설명](#) 참고)
- `int32`, `float32` 등 상황에 맞는 타입을 골라 씀

## 2. pandas가 뭐야?

**표(엑셀 같은 형태) 데이터를 다루는 라이브러리**예요. numpy 위에 만들어졌어요 (numpy를 내부적으로 사용).

## 3. DataFrame이 뭐야?

pandas에서 쓰는 **"표" 자료구조**예요. 엑셀 시트를 떠올리면 돼요.

```python
import pandas as pd

value = [[32, 68, 220, 72],
         [28, 95, 180, 45],
         [38, 88, 210, 60]]
df = pd.DataFrame(value)

print(df)
#     0   1    2   3
# 0  32  68  220  72
# 1  28  95  180  45
# 2  38  88  210  60

print(df[0][1])   # 0번 열, 1번 행 값 -> 28
```

- 행(row) = 가로줄 (데이터 한 개 단위, 예: 사람 1명 정보)
- 열(column) = 세로줄 (속성 하나, 예: 나이, 키, 몸무게)

### 언제 쓰나?
- CSV, 엑셀 파일 읽고 쓸 때
- 데이터 정리/필터링/정렬할 때 (예: "나이 20살 이상만 뽑기")
- 여러 컬럼(속성)이 있는 데이터를 다룰 때

---

## 4. numpy vs pandas 비교

| | numpy | pandas |
|---|---|---|
| 기본 단위 | 배열(array) | 표(DataFrame) |
| 잘하는 것 | 숫자 계산, 행렬 연산 | 데이터 정리, 조회, 필터링 |
| 이름 | 열 번호만 있음 (0,1,2..) | 열에 이름 붙일 수 있음 ("나이", "이름" 등) |
| 비유 | 계산기 | 엑셀 |

**관계**: pandas의 DataFrame은 내부적으로 numpy 배열을 사용해요. 그래서 두 라이브러리는 세트로 같이 쓰이는 경우가 많아요.

```
pandas (표 정리)
   └─ numpy (숫자 계산 엔진)
```

## 5. AI 분야에서는 어떻게 쓰여?

### numpy
- **이미지 데이터 자체가 numpy 배열**이에요. 예를 들어 카메라로 찍은 사진 한 장은 `(높이, 너비, 채널)` 형태의 numpy 배열로 저장돼요.
  ```python
  img.shape  # (480, 640, 3)  -> 세로 480, 가로 640, 컬러채널(RGB) 3
  ```
- 자율주행차 프로젝트(`autocar`)에서 카메라로 받은 영상을 처리(윤곽선 찾기, 색상 분석 등)할 때 numpy로 픽셀 값을 직접 계산해요.
- 딥러닝 모델(텐서플로우, 파이토치)도 내부적으로 numpy와 비슷한 배열 구조(텐서)를 씀 -> numpy 문법을 알면 이해가 쉬워짐

### pandas
- AI 모델을 학습시키기 전, **학습용 데이터를 정리하는 단계**에서 필수로 써요.
  - CSV 파일로 된 학습 데이터를 불러오기
  - 빈 값(결측치) 제거하거나 채우기
  - 필요한 컬럼(속성)만 골라내기
  - 데이터를 학습용/테스트용으로 나누기
- 예: "지난 주행 기록 데이터(속도, 방향, 센서값)"를 CSV로 저장해놨다가 pandas로 불러와서 정리한 다음, 그 값을 numpy 배열로 바꿔서 모델에 넣어요.

### 정리하면 이런 흐름이에요

```
데이터 파일(csv 등)
   │  pandas로 읽고 정리 (DataFrame)
   ▼
숫자 배열로 변환
   │  numpy 배열 (array)
   ▼
AI 모델 학습/예측 (모델에 입력)
```

즉, **pandas = 데이터 정리 담당**, **numpy = 계산 담당**이라고 기억하면 편해요.
